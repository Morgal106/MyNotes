## 单点定位
- 标准单点定位 SPP
	- 公式->泰勒展开->矩阵
	- 动态 SSP
- 精密单点定位 PPP

|            | 标准单点定位 | 精密单点定位 |
| ---------- | ------------ | ------------ |
| 观测值     | code         | phase        |
| 星历       | 广播星历     | 精密星历     |
| 精度       | 低           | 高           |
| 解算复杂度 | 低           | 高           |
| 实时性     | 强           | 弱           | 
- 精度评定参数 DOP
	- 种类
		- PDOP x,y,x
		- TDOP t
		- GDOP x,y,z,t
		- HDOP B,L
		- VDOP H
	- 值越小精度越高

|      | best  | very good | good | medium | pass | bad |
| ---- |:-----:|:---------:|:----:|:------:|:----:|:---:|
| PDOP | 0.5-2 |    2-4    | 4-6  |  6-9   | 9-20 | >20 |
